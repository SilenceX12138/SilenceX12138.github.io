MATLAB-Chapter5-图形绘制

Posted on 2020-12-04 In AI , MATLAB Valine:

Chapter5 图形绘制

全文共2134字，推荐阅读时间15~20分钟。

二维图形

基本操作

  • plot

    • 默认按照蓝色实线绘制，并做平滑处理。

    • 统一x轴绘制

      纵坐标行向量需要转置变成列向量

  • plotyy：双y轴绘图，绘制横坐标相同，但是纵坐标对应的变量量纲、范围等不同的图形。

    • Y1默认在左侧，Y2默认在右侧。

    • plotyy不能够直接对曲线属性进行设置，需要使用句柄图形控制完成。

      • AX(1)和AX(2)对应两侧坐标轴的句柄，表示坐标轴属性。

      • H1和H2是对应的曲线的句柄，可以根据函数的返回值进行操作。

  • semilogx：x轴取对数（10为底）的绘图

  • semilogy

  • loglog：两个维度上都取对数

    • logspace：对数空间中，以10为底数生成值（和linspace对应）。

  • polar：极坐标图

    • theta是弧度制

修饰

线型

  • -：实线
  • --：虚线
  • :：点线
  • -.：点划线
  • none：无线

颜色

  • b：蓝色
  • g：绿色
  • m：红紫色
  • w：白色
  • c：蓝绿色
  • k：黑色
  • r：红色
  • y：黄色

标记符号

对曲线上有值的点进行标记

  • *：星号
  • x：叉号
  • s：方块
  • p：五角星
  • ^/>/./O/d/h/v/<

图形属性

  • LineStyle
  • LineWidth
  • Marker：数据点标记符号
  • MarkerSize：标记符大小
  • MarkerEdgeColor：标记符边缘颜色
  • MarkerFaceColor：标记符填充颜色

文字标注

  • title(s)：图标题

  • xlabel(s)：横坐标名

  • ylabel(s)

  • legend(s, position)：在指定位置建立图例，按照绘图函数的参数顺序生成。

    • s：图例中的文字注释，可用s1,s2,...表示。
    • position：指定图例位置
      • 0：自动最佳位置（不覆盖原图）
      • 1：右上角（默认）
      • 2：左上角
      • 3：左下角
      • 4：右下角
      • -1：图右侧

  • text(x, y, s)：在x轴和y轴的指定位置添加文字注释

    x和y是值列表

重叠绘图

  • hold on：保持当前图形
  • hold off：取消当前图形的保持功能
  • hold：双向切换开关（对当前状态取反）

多子图绘制

  • subplot：在一图形中同时显示多幅独立的子图

    • 所有子图彼此独立，绘图命令可以在子图中使用。

      注意和legend中的位置进行区分

    • 非等比例子图

多窗口绘图

  • figure(n)：创建绘图窗口，n是窗口顺序号。

    figure1不需要单独指定

坐标轴控制

  • axis

网格线与坐标框

  • grid
    • grid on
    • grid off
    • grid
  • box：确定当前坐标框是否呈封闭形式（省略上框线和右框线）
    • box on
    • box off
    • box

特殊情况

  • bar：垂直条形图

    *和.*的区别

    • 对于矩阵运算来说，*是标准的矩阵点乘。

    • 对于矩阵和数字的运算来说，二者没有区别。

  • barh：水平条形图（横向是Y轴）

  • errorbar：误差图

  • area：面积图，即填充曲线的积分面积。

  • hist：柱形图，用于显示数据分布情况。

    柱形图和条形图的区别

    • 条形图用来表达数据间的比较关系，例如不同产品的销售情况。（自变量是产品种类）
    • 柱形图用来展示数据的分布或趋势变化，自变量通常是量化数据。
    • 柱形图不同柱一般不能重新排序，而条形图不同条可以任意重新排序。
    • 柱形图各个柱之间通常没有空隙
    • 柱形图的纵轴通常是横轴元素的计数结果

  • stem：火柴杆图

  • stairs：阶梯图

  • pie：扇形图

  • compass：罗盘图

  • feather：羽毛图

三维图形

基本操作

  • plot3：调用格式和plot一致

    • ezplot3：对符号函数进行绘制

  • mesh：网格图函数

    • ezmesh：符号函数的网格图绘制

  • surf：对网格进行填充

    • ezsurf

  • meshgrid：生成二元函数$z=f(x,y)$中$X-Y$平面上的句型定义域中的数据点矩阵$X$和$Y$.

特殊情况

  • bar3
  • bar3h
  • cylinder
  • sphere
  • stem3
  • contour3
  • waterfall
  • pie3
  • surfnorm

修饰

  • view：控制三维图形的观察点和视角

    • AZ：方位角，角度制。

      绕Z轴顺时针为正，逆时针为负。

    • EL：仰角

  • colormap：图形颜色

  • alpha：透明度

    • 透明度值越大，图像越不透明。
