全文共620字，推荐阅读时间7~10分钟。

概述

  • 条件表达式用于在filter/annotate/aggregate/update中实现if逻辑。

  • 取以下模型为例

When

  • class When(condition=None, then=None, **lookups)

    • When接收返回布尔值的表达式，根据then确定返回值。

Case

  • class Case(*cases, **extra)

    • Case根据传入的When表达式选择返回值（类似switich-case语句）

      Case实际上就是在组织多个When
