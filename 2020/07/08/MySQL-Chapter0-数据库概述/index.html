MySQL-Chapter0-数据库概述

Posted on 2020-07-08 In Web , MySQL Valine:

全文共348字，推荐阅读时间3~5分钟。

背景

  • 应用程序不需要自己管理数据，而是通过数据库软件提供的接口来读写数据。

数据模型

层次模型

  • 树形结构

网状模型

  • 图结构

关系模型

  • 二维表格
  • 通过行号和列号定位
  • 占数据库的主导地位

数据类型

  • 关系表中，除了每一列的名称，还需要定义每一列的数据类型。
Name Type Description
INT 整型 4字节
BIGINT 长整型 8字节
REAL 浮点型 4字节
DOUBLE 浮点型 8字节
DECIMAL(M,N) 高精度小数 M表示数字总共多少位，N表示小数有多少位，常用于财务计算。
CHAR(N) 定长字符串 N表示字符串长度
VARCHAR(N) 变长字符串 N表示长度上限
BOOLEAN 布尔类型 True/False
DATE 日期类型 2020-07-01
TIME 时间类型 12:20:25
DATETIME 日期和时间类型 2020-07-01 12:20:25

BIGINT和VARCHAR是使用最广泛的两种类型

语法特点

SQL关键字不区分大小写，但是规范如下：

  • 关键字统一使用大写
  • 表名和列名统一使用小写
