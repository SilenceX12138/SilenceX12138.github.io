智能手机系统

背景

  • 传统功能机
  • 智能移动终端
    • 智能指配备了OS,并不是人工智能。（扩展性）

平台

IOS

定义
  • 硬件到服务的闭环

  • 不授权在非Apple硬件安装

  • Mac OS & IOS

分层架构
  • Core OS(C)：基于微内核
  • Core Services(C)
  • Media(C&Object-C)
  • Cocoa Touch(Objective-C)：提供应用程序框架和API
开发环境
  • Xcode
  • Objective-C&Swift

Android

定义
  • 基于Linux
  • 接口基于Java
层次

  • Linux Kernel：底层硬件的管理

  • Libraries

    • 大部分由C/C++编写，实现了通用功能。

    • Android Runtime：装入了一个虚拟机，每个应用有自己独立的虚拟机。

      .dex：将.class进行字节码再次翻译达到压缩目的

  • Application Framework

    • 基本由Java编写
    • 提供开发使用的API
    • 应用程序可以使用JNI调用下层函数库（高效性）

  • Application

    • 平台无关
    • Java开发
Android Market(Google Play)
  • Market本身自己也是应用
开发环境
  • Android SDK
  • Java SDK
  • 集成开发环境

Windows CE/Mobile

  • Windows CE
    • 嵌入式系统
    • 支持ARM/MIPS/x86/SH4
  • Windows Mobile
    • 基于Windows CE

Symbian

开发生态并不友好，赶走了开发者，导致了灭绝。

区块链平台操作系统

区块链

  • 链式数据结构
    • 记录前一个结点的地址和哈希值
  • 分布式节点共识算法
  • 智能合约

不可篡改、可追溯、构建信任、去中心、支持智能合约。

发展

1.0

2.0（以太坊）

  • 支持图灵完备的虚拟机
  • 使用智能合约程序管理资产

3.0

  • 在货币和金融外，在其他产业领域上提供可扩展的区块链应用。
  • 联盟链(ToB)

应用

  • 不可篡改
    • 物流、法证、电子政务、发票、学位证书
  • 共识信任
    • 供应链金融
  • 去中心
    • 新型交易
  • 确权和可信
    • 医疗、物联网、大数据协调

Reference

  • BUAA OS Lecture, Spring Of 2020.
