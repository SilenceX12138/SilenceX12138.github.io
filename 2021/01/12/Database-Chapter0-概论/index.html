Chapter0 概论

数据和信息

  • 信息是一种已经被加工为特定形式的数据
  • 数据是信息的载体，是信息的具体表现形式。

处理

  • 对数据进行加工得到信息的过程

管理

  • 数据处理的基本操作
  • 数据库技术就是一种数据管理技术
    • 提供数据共享

数据库系统

  • 建立在DBMS之上，以数据管理为主要功能的计算机软件系统。

DBS > DBMS > DB

  • DBS包括了用户/应用程序构成的交互层

架构(B/S)

  • Browser
  • Server
    • Web Server
    • DB Server(DBMS location)

原理

  • 数据模型
  • 底层实现技术、事务处理、数据库安全、完整性控制
  • 关系数据理论、数据库设计方法

事务处理（原子性）是DBMS的工作，不是DBS.

实例

  • 图书馆
  • 电商

Reference
